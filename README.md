OneGo is an all-in-one daily utility mobile app built using a no-code platform.

## Problem Statement
Users often rely on multiple apps for travel, nearby places, emergency contacts,
and daily information. OneGo combines these features into a single app.

## Features
- Nearby places information
- Emergency contacts
- Travel & daily utility support
- Simple and user-friendly interface

## How it was built
- No-code / low-code platform
- UI and app flow designed manually
- No programming language used

## Status
Prototype / MVP

## Future Improvements
- Login system
- Location-based recommendations
- More utility services
